Operators:
	Assignment
	Arithmetic
	Relational
	Logical
	Null
	Misc - sizeof(), typeof()

program elements:
	Control Statements : execution path of a program
	Selection statement - if, switch
	Iteration statement - while, do-while, for, ea
	Jump statements - return, break, continue,  goto, throw

Goto -  used to transfer control within function
	identified with a valid name followed by a ':'
	unstructured control flow of statements

eg:

switch(intv1, strv2, boolv3){
	case(1, "hi", True):
		write();
}
