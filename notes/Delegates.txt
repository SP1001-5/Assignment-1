Delegates
	is an object that refers to a method, Holds reference to methods of a class
	Provides a good way to encapsulate the methods
	type safe pointer of any method
	delegates are used to implement call back and even driven program
	delegates can be chained together as 2 or more methods called or an event
	Delegates are used to invoke anonymous methods

Note :	delegate/s can point to method/s which agree on the signature (matching signature)
