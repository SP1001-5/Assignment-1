Various types of language : 
1.Assembly Language:
	Low level
	directly communicate hardware resources
	-> Assembler : program translates code into machine understand language (0 and 1)
	  Syntax : basic structure
	  Label : a word, number, symbol to referebce point to locate the instructor
	  command : Order
	  Operand : data manipulator
	  Directive : instruction for settings, changes or to perform actions
	  Macro : Short cut representation of a sequence of instructions
	  Nemonincs : represent abbreviations of the action being performed
	eg : MOV a,10
	     ADD a,b,c
2.Machine Language:
	00000000 - stop the program
	00000001 - start the program
	uni code - 16  bits
3.Programming language:
	-> procedural language - Sequence of instruction   
	eg : C, C++, Java, Basic, Pascal
	-> Function Programming Lang - More focus on the output of mathematical calculations
	eg : scale, trlang, Python, F#
	-> OOP Lang -	treats a program as a group of object, composed of data and program elements (attribute and methods)
	eg : C++, Java, C#, Python, Ruby
	->Scripting Lang - Automates repetitive tasks and manage dynamic content
	eg : PHP, javascript, Perl, Python, Bash , Node.js
	-> Logic Programming Lang : Instead of passing instructions of what to do , we can express a series of facts and rules that interpreted by the system and gives out decision
	eg : Prolog, Datalog, alma, Absys
	-> Front end lang : mostly used for supporting  UI/UX	
	eg : HTML/CSS, JavaScript, react.js, Anguar	
	-> Back end lang : deals woth storage and manipulation of data without direct interaction with the user
	eg : javaScript, PHP, Java, C#
