CSharp : a general purpose, modern, OOP Language developed by Microsoft Anders Hejisberg and his team 

Namespace : Collection of class, interfaces, events, delegates, Structures other namespace
		logical way to organize our program contents
		it help in avoiding name clashes

Data Types : Basically either 
- Value type : All integral data types
		are derived from System.ValueType
		are stored in Stack memory
	types:
		->Predefined -> all primitive
		->User defined -> Structure
					-> Enum
- Reference type : Stores address like objects, Arrays
		are stored in heap memory
	types:
		->predefined -> Object
					-> string
		-> user defined ->Array
					-> Classes
