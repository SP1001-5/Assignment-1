Inheritance
	-> Extensions of the capabilities of a class
TYpes:
  
  1.SIngle inheritance
  2.Multilevel Inheritance
  3.Hybrid Inheritance

Type of relation:
  1. 'is a' relation (inheritance)
	class dog(){}  -> class Labrador : dog(){}  Lab is a dog

  2. 'has a' relation (aggregation/Composition)
	class Engine(){
		int EngineId;
		double Horsepower;
	}
	class Car(){
		Engine engine;
		String Make;
		double Cost;
	}					    Car has a Engine
