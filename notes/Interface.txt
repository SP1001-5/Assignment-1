Interface

	-> like classes that can contain method, properties, events, delegates but not only declarations
	->interface members are public by default 
	->the class that implements an interface will have to provide implementation for all the methods 
	-> they do not allow explicit access modifiers 
	-> a class can implement more than one interface
	-> One interface can implement other interface
	->An implementing class should define all the methods in the interface chain (one interface implementing many other interface)
	-> cannot instantiate interface, but a reference objects can be created
