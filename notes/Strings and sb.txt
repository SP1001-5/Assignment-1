Strings
->collection of characters
->sealed class
->allocated in heap memory
->immutable
->string memory cannot be changed dynamically

string s1 ="hello"
s1 -> stores address of the value "Hello" in string pool

s1 = "hello world"
s1 -> stores the address of hello world and that hello will not be deleted

string s2 = "hello"
s2-> s2 will now point the address of the hello which is frst created by s1

string str = new string(char[] ch);
str-> will point the new memory address other than storing the value in pool

String Builder:
	same as strings but mutable
